# GitHub PRのイベント

|イベント名|説明|
|:--:|:--:|
|opened|プルリク作成|
|closed|プルリクclose/ブランチマージ|
|reopened|プルリク再open|
|edited|タイトル・body編集/baseブランチ変更|
|assigned|担当追加|
|unassigned|担当除外|
|review requested|レビューリクエスト|
|review requested removed|レビューリクエスト取り下げ|
|labeled|ラベル追加|
|unlabeled|ラベル除外|
|synchronized|ブランチにpushされたコミットをプルリクに反映|
